Admonitions

Package Name
fifthtry.github.io/admonitions

The admonitions package(fifthtry.github.io/admonitions). Use FTD custom admonitions styles for your web pages.

How to use?
Add this to your FASTN.ftd
-- fastn.dependency: fifthtry.github.io/admonitions as cbox

-- fastn.auto-import: cbox

cbox.note Basic usage example

Component Name
-- cbox.note:
Use below code snippet to use this admonitions component as cbox.
How to use
-- cbox.note: Noteworthy Heading

Of all the things possible, this is what you came up with? I am disappointed in
your lack of ingenuity.

-- end: cbox.note

Output of above example:

🗒️
Noteworthy Heading
Of all the things possible, this is what you came up with? I am disappointed in your lack of ingenuity.
